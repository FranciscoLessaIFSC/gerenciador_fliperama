# 🕹️ ArcadeManager

> Sistema completo de gerenciamento de fliperama com painel administrativo e área do cliente.

---

## 📋 Sobre o Projeto

O **ArcadeManager** é uma plataforma web desenvolvida para modernizar e facilitar a gestão de fliperamas. O sistema oferece dois ambientes distintos: um painel administrativo robusto para os operadores do estabelecimento e uma interface intuitiva para os clientes aproveitarem ao máximo sua experiência.

---

## ✨ Funcionalidades

### 👤 Área do Cliente

- **Cadastro e login** com autenticação segura
- **Carteira virtual** — recarga de créditos e consulta de saldo
- **Catálogo de máquinas** disponíveis no estabelecimento
- **Histórico de partidas** e pontuações
- **Ranking geral** e por máquinas

### 🛠️ Área do Administrador

- **Dashboard** com visão geral do estabelecimento em tempo real
- **Gerenciamento de máquinas** — cadastro, edição, ativação/desativação e manutenção
- **Controle financeiro** — receita por máquina, relatórios diários/mensais e exportação de dados
- **Gestão de usuários** — listagem, bloqueio e consulta de perfis de clientes
- **Controle de créditos** — recarga manual e estorno
- **Configuração de promoções** e eventos temáticos
